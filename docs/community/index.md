# :material-account-group: Community

SRv6.md is a community-driven project. Everyone is welcome to contribute, whether you're fixing a typo or writing an entire section.

## How to Contribute

1. **Fork** the [GitHub repository](https://github.com/eldaninavas/srv6.md)
2. **Create** a branch for your changes
3. **Write** your content in Markdown
4. **Submit** a Pull Request
5. **Review** - maintainers will review and merge

[:material-arrow-right: Full Contributing Guide](contributing.md){ .srv6-btn .srv6-btn-primary }

## Ways to Help

- :material-pencil: **Fix typos** or improve existing content
- :material-file-plus: **Add new pages** for missing topics
- :material-flask: **Share lab configs** that others can reproduce
- :material-bug: **Report issues** or suggest improvements
- :material-translate: **Translate** content to other languages
- :material-star: **Star** the repo to show support

## Code of Conduct

We follow a simple rule: **be kind and constructive**. This project welcomes contributions from everyone regardless of experience level.
